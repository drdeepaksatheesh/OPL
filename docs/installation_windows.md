\# Windows Installation Guide



\## OpenPhysiologyLab v0.1-alpha



OpenPhysiologyLab v0.1-alpha is currently a source-run Python application. It does not yet have a one-click installer.



There are two ways to install it on Windows.
After installation, see docs/launching_openphysiologylab.md to open the app.



\---



\## Option 1: Manual installation



Use this if you are comfortable following written steps.



Read:



```text

docs/installation\_windows\_manual.md

```



This guide explains how to:



\* install Python

\* download OpenPhysiologyLab

\* open Command Prompt

\* install required Python packages

\* launch the app



\---



\## Option 2: ChatGPT-assisted installation



Use this if you are new to Python, Command Prompt, Git, or software installation.



Read:



```text

docs/installation\_windows\_ai\_assisted.md

```



That file contains a prompt you can copy and paste into ChatGPT or another AI assistant.



The AI assistant will then guide you slowly, one step at a time.



This is recommended for first-time users.



\---



\## What you need



For the software:



\* Windows 10 or Windows 11

\* Python 3.10 or 3.11

\* internet connection for installing Python packages



For ECG-like recording:



\* NPG Lite - For details about the NPG Lite Beast Pack used during development, see docs/hardware_npg_lite.md. For firmware upload instructions, see docs/firmware_flashing_npg_lite.md.

\* USB cable

\* compatible USB serial firmware uploaded to NPG Lite

\* ECG gel electrodes



\---



\## Safety note



OpenPhysiologyLab is not a diagnostic medical device.



Do not use it to diagnose disease, guide treatment, replace certified ECG machines, or make emergency medical decisions.



Use it for teaching, experimentation, validation, and understanding physiological signals.
